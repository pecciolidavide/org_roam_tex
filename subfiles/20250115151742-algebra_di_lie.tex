\section{Algebra di Lie}
\label{sec:org2803c06}
\begin{definizione}
Uno \href{20241205142027-spazio_vettoriale.org}{spazio vettoriale} \(V\) dotato di un’ulteriore operazione \([\,\cdot, \cdot\,]: V\times V \to V\) tale che:
\begin{enumerate}
\item \([v, w] = −[w, v]\) (anticommutatività);
\item \([au + bv, w] = a[u, w] + b[v, w]\) (linearità);
\item \([u, [v, w]] + [v, [w, u]] + [w, [u, v]] = 0\) (identità di Jacobi),
\end{enumerate}
è detto \uline{algebra di Lie}.
\end{definizione}
