\section{Gruppo lineare}
\label{sec:orgb798a1f}
Sia \(\K\) un \href{20241205142049-campo.org}{campo}. Il \textbf{gruppo lineare} di dimensione \(n\) è
\begin{equation*}
\operatorname{GL}(n, \K) \coloneqq \set{A \in \K^{n,n}: \operatorname{det}A\neq 0 }
\end{equation*}
(vedi \href{20250104111539-spazio_delle_matrici.org}{Spazio delle matrici} e \href{20250104111751-determinante_di_una_matrice.org}{Determinante di una matrice})

È un \href{20241205141146-gruppo_abeliano.org}{gruppo}.
