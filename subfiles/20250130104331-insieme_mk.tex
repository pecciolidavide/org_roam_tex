\section{Insieme per MK}
\label{sec:org691e48b}
Si è nell'ambito della \href{20250130104245-morse_kelly_set_theory.org}{Morse Kelly Set Theory}.

Una \href{20250130104320-classe_mk.org}{classe} \(A\) è un \textbf{insieme} se e solo se esiste una \href{20250130104320-classe_mk.org}{classe} \(B\) tale che
\begin{equation*}
A \in B
\end{equation*}

La \(\mathcal{L}_{\in}\)-\href{20250131103317-formula_del_prim_ordine.org}{formula} ``\(x\) è un insieme'' è
\begin{equation*}
\operatorname{Set}(x):\qquad \exists y\ (x \in y)
\end{equation*}
