\section{Collegare le note - Zettelkasten}
\label{sec:orgbd84dc9}
In un sistema \href{20251112215403-zettelkasten.org}{Zettelkasten} un buon modo iniziale per stabilire quali collegamenti tra le note sono da fare è il seguente:
\begin{quote}
idea compass:
\begin{itemize}
\item N - what larger pattern does this concept belong to?
\item S - what more basic components is this concept made of?
\item E - what is this concept similar to?
\item W - what is this concept different from?
\end{itemize}
\end{quote}
come spiegato in .

Questo può essere fatto anche in ambito matematico, anche se con qualche aggiunta:
\begin{itemize}
\item S' - da quali fatti segue logicamente il teorema/lemma/proposizione?
\end{itemize}
